# 五人找朋友打法分阶指南

这份文档不是解释 AI 怎么算牌，而是把这类牌局整理成一条适合人类阅读和练习的学习路线。

如果你刚接触这套五人找朋友，可以把它理解成：

- 初级阶段：先做到不乱打，知道每一手为什么这么出。
- 中级阶段：开始会留牌、会配合同伴、会考虑下一轮牌权。
- 高级阶段：开始会读局、会算结构、会围绕升级目标做整局布局。

配套规则请先看 [rules-summary.md](rules-summary.md)。

文中常说的 `找线`，意思是找一条能把牌权接起来、让己方连续出牌的路线；`断线` 则是把这条路线切断，让对手接不上后续节奏。

## 1. 先记住这 6 条总原则

- 你的目标不是只赢这一轮，而是让自己的阵营更容易升级。
- 长期通关目标会一路经过 `A 级`、`王级`，再往更高的 `+2 / +A` 级别推进；越往后，整局布局价值越大。
- 主牌是最宝贵的控制资源，不要随手浪费。
- 对子、拖拉机、火车、推土机这类结构牌，价值通常高于一时顺手。
- 朋友没站队之前，打家方和闲家方都不能只看眼前得失，要先判断阵营。
- 分牌不是见分就吃；很多时候“谁拿这一轮”和“谁保留牌权”更重要。
- 扣底是大 swing，后程很多看似普通的出牌，其实都在为末局保底或抢底铺路。
- 在 `J / Q / K / A` 这些特殊级里，级牌扣底往往比普通升级更值钱，因为把打家打下去本身就等于推进自己的长期进度。

## 2. 初级打法：先做到稳，不乱送

初级阶段的目标不是“打得花”，而是“尽量少犯明显错误”。

### 2.1 初级阶段最重要的目标

- 先学会按规则跟牌，不漏牌、不乱拆。
- 学会分清什么牌该留，什么牌可以顺手处理。
- 不因为一时想赢，就把整手主牌和大牌打散。
- 先建立“打家 / 朋友 / 闲家”的基本立场。

### 2.2 初级玩家最该养成的出牌习惯

- 能不拆对子，就先别拆对子。
- 能不拆拖拉机，就先别拆拖拉机。
- 主牌数量不多时，不要轻易拿主去垫普通小轮。
- 看不清局势时，优先出自己较弱花色的小牌，而不是先扔关键高牌。
- 还没确定朋友是谁时，打家不要太早把全套好牌亮干净。
- 闲家看到打家明显在找朋友时，不要只顾抢眼前一手，要想着怎么断掉他的节奏。

### 2.3 初级阶段的打家怎么打

- 刚开打时，先想清楚自己是“要找朋友”还是“已经能自己控几轮”。
- 如果你的主牌并不厚，不要急着连续吊主，先把朋友找出来更重要。
- 找朋友的那门，如果你自己后面还有延续牌，这门通常更值得主动发动。
- 你手里如果有一门很长、而且有大牌衔接，这门往往是最适合组织节奏的门。
- 如果你底牌里埋了很多关键分牌，后面更要保住主牌，不要轻易争无意义的小轮。

### 2.4 初级阶段的闲家怎么打

- 第一目标不是“每轮都抢”，而是别让打家方轻松连着拿牌权。
- 打家找朋友时，重点看他先出哪一门、为什么出这门。
- 如果你同门不多，先记住自己这门以后可能有毙牌机会。
- 自己没把握时，不要轻易拿大主去压一轮不大的牌。
- 后程一旦发现打家方主牌快见底，就要开始考虑抢末局和扣底。
- 如果当前正打在 `J / Q / K / A` 这类特殊级，而且你手里同时有级牌、长主和大主回手，那么“抢级牌扣底”往往比单纯去追普通分更值得。

### 2.5 初级阶段最常见的 6 个错误

- 有对子先拆对子，导致后面别人出对时只能吃亏。
- 小分轮也上主，结果关键大轮没主可用。
- 打家一上来就把找朋友路线和主控能力都打没了。
- 闲家只顾抓眼前分，没去破坏打家找朋友和组织节奏。
- 明明该保结构，却因为“这张看起来最大”就先出了。
- 末局前没有保留能守底或冲底的关键主牌。

### 2.6 初级阶段怎么练

- 每打一手前先问自己一句：这手打完，我还剩什么结构？
- 每一轮结束后回看：我刚才是不是为了一点小便宜拆了大结构？
- 先别追求复杂甩牌，先把“跟牌正确、少拆结构、少乱上主”做好。

## 3. 中级打法：开始经营牌权和队友协同

中级阶段的核心，是从“这手怎么出”升级到“这手出完以后，下一轮会怎样”。

### 3.1 中级阶段最重要的目标

- 学会经营牌权，不轻易把先手送回去。
- 学会判断什么时候该保结构，什么时候该主动拆牌换节奏。
- 学会围绕朋友是否站队，切换打法重点。
- 学会把“抓分、清主、找朋友、保底”放在同一盘棋里看。
- 学会判断什么时候该把“级牌扣底”提到和升级同级，甚至更高的优先级。

### 3.2 中级玩家要开始会的 5 件事

- 会留后手：不是单看当前最大，而是看赢下这一轮后还能不能继续组织。
- 会保主控：主牌不仅是拿来毙人的，更是决定谁能掌握节奏的资源。
- 会分阶段：朋友没站队和朋友已站队，是两种完全不同的局面。
- 会打长门：一门牌如果能连着出，往往比单张零碎高牌更有价值。
- 会算风险：有些分牌虽然能吃到，但代价是把主和结构一起送掉。

### 3.3 中级阶段的打家怎么打

- 开局先判断自己是“主控型”还是“副牌组织型”。
- 如果主牌明显厚，可以考虑先清主，减少后续副牌被毙的风险。
- 如果主牌不够厚，重点不在硬清主，而在尽快把朋友带出来并连上线。
- 找朋友之后，打法要立刻切换，不要还像单打独斗那样继续试探。
- 当朋友已经站队，而且你们某一门有长度优势时，要尽量把这门打成长节奏。
- 如果你埋底压力很大，后程要优先保能守底的那几张关键主牌。

### 3.4 中级阶段的朋友怎么打

- 朋友最怕的不是“没抓到分”，而是站错时机或站出来后断线。
- 如果打家叫的是某门大牌，不一定非要第一时间站队，关键看站出来后能不能继续配合。
- 一旦已经站队，目标就从“表明身份”变成“配合打家把节奏接住”。
- 有时候朋友该做的不是抢大，而是让打家继续先手，把牌型打顺。
- 如果你能判断打家下一轮想出哪一门，就尽量别提前把那门节奏打乱。
- 反过来，闲家看到朋友已经在这一墩用朋友牌站出来时，如果自己手里也有同一张朋友牌、但这张牌并不能反超当前赢家，通常不该为了“跟大牌”把它白白送掉；更像人类的处理是先跟同门低张，把这张高牌留给后面真正能换牌权的时机。

### 3.5 中级阶段的闲家怎么打

- 闲家最重要的是切断打家和朋友的连线。
- 打家在找朋友时，你要看的是：他为什么出这门，他想让谁接。
- 如果打家方在某一门开始形成长节奏，闲家要尽快想办法用主牌或断门去打断。
- 朋友一旦站队，闲家三家就要把重点从“找朋友判断”转到“压主控、抢末局、争扣底”。
- 中后程当分数不够时，别只想平分，要主动争那些可能通向底牌的大轮。

### 3.6 中级阶段最重要的三个判断题

- 我现在赢这一轮，下一轮还有没有继续出牌的能力？
- 我这手如果拆结构，是换来了节奏，还是只是图一时舒服？
- 当前这门打下去，是更利于我方连线，还是更利于对手断门上主？

### 3.7 中级阶段常见误区

- 朋友已经站队了，还在按“找朋友试探局”去打。
- 明明该保住牌权，却为了吃一点分硬拿大牌去抢。
- 只会清主，不会判断什么时候不该清主。
- 看到自己有大牌就急着兑现，没有先看队友和下一轮的接法。
- 没有把“守底”提前到中后程考虑，总是等最后几轮才临时补救。

### 3.8 中级阶段怎么练

- 每次拿到先手时，先想两步：这手出完，下一轮谁最可能再先出？
- 每次朋友站队后，复盘自己有没有及时换打法。
- 每局结束后单独回想一个问题：本局真正的转折点，是哪一手牌权没接住？

## 4. 高级打法：读局、控节奏、围绕升级去打

高级阶段不只是“记牌更多”，而是开始真正理解这桌牌为什么会往这个方向走。

### 4.1 高级阶段最重要的目标

- 根据已出牌和玩家动作，持续修正每个人的门数、结构和身份倾向。
- 不只看这一门强不强，而是看整局节奏会落到谁手里。
- 在抓分、跑分、保底、断线、找朋友之间做取舍。
- 让自己的每一手牌都服务于整局升级目标，而不是只服务于这一轮。

### 4.2 高级玩家要具备的核心能力

- 会记牌：至少要大致记住主牌、关键大牌、成结构的高牌都出了多少。
- 会读人：延迟站队、硬垫、故意不压、提前清某门，这些动作都在传递信息。
- 会控节奏：知道什么时候该加速，什么时候该故意放慢。
- 会做取舍：有些分必须放，有些轮必须送，有些结构必须保。
- 会布局末局：真正的高手往往从中盘就开始为最后扣底做准备。

### 4.3 高级阶段的打家怎么打

- 先决定这一局是“稳控升级局”还是“高风险冲穿局”。
- 如果主控足够厚，重点是把闲家的断门毙牌机会压到最低。
- 如果主控一般，但副牌路线很好，就别把局打成纯主牌消耗战。
- 找朋友时要考虑的不只是“谁会站队”，还要考虑“站队后能不能形成连续牌权”。
- 当闲家明显在准备抢底时，打家要敢于提前牺牲一些中盘小分，换末局控制权。

### 4.4 高级阶段的朋友怎么打

- 高级朋友不只是“帮打家抓分”，而是要判断打家这一局真正缺什么。
- 有时打家缺的是一门延续牌，你就要接这门。
- 有时打家缺的是主控喘息空间，你就要替他挡一轮主。
- 有时打家需要的是末局有人接应，你就必须保留那几张关键牌，不提前兑掉。
- 真正好的朋友打法，是让打家觉得节奏一直在自己手里。

### 4.5 高级阶段的闲家怎么打

- 高级闲家不是“谁能压谁就压”，而是三家一起逼打家犯选择错误。
- 要主动制造这种局面：打家无论怎么选，都要付出代价。
- 如果看出打家想保底，就逼他提前交主。
- 如果看出打家想清主，就尽量在副牌上制造断门威胁。
- 如果朋友已经亮明，而且两家快要连上线，闲家就要果断把资源集中到断线和抢末局上。

### 4.6 高级阶段最值钱的判断

- 哪一门现在看起来最大，但其实一出就会把节奏送回去？
- 哪一手虽然当下不赚分，却能让己方两轮后更容易接住？
- 哪个玩家现在最像在“留着末局牌”，能不能提前逼出来？
- 这一轮如果让同伴先大，整体收益会不会比自己硬抢更高？

### 4.7 高级阶段常见误区

- 记了很多牌，却没有把记牌转化成出牌取舍。
- 过分追求每一轮都对，反而错过真正决定胜负的节奏点。
- 太迷信“清主一定好”，忽视了副牌长门和朋友连线的价值。
- 过度炫技式甩牌，结果把自己的结构和保底资源一起甩没。
- 只算自己手牌，不算别人为什么这么出。

### 4.8 高级阶段怎么练

- 复盘时不要只看“输了哪轮分”，要看“哪一轮后节奏完全变边了”。
- 每局结束后，尝试口头复述一遍：谁什么时候站队，谁什么时候断哪门，谁什么时候开始保底。
- 练习给每个关键动作找原因，例如“他为什么这轮不压”“他为什么先出这门小牌”。
- 需要把问题局发给别人继续复盘时，优先直接附上日志里的 `11` 位回放种子和 `169` 位开局码；开局码区分大小写，这只是帮助复现牌局，不替代对打法本身的复盘。

## 5. 三个阶段最直观的区别

- 初级看当前：这手怎么合法、怎么顺手。
- 中级看下一轮：这手打完以后，谁还能继续组织。
- 高级看整局：这手是否在为升级、保底和阵营节奏服务。

## 6. 给普通玩家的最短练级路线

如果你只想快速从“会打”提升到“打得像样”，按这个顺序练最有效：

1. 先练少拆对子、少乱上主、少送关键大牌。
2. 再练拿到先手后先想下一轮，别只看这一轮。
3. 接着练朋友站队前后切换打法。
4. 最后练记主牌、记关键大牌、记谁断门。

## 7. 一句话收尾

这类牌真正的水平差距，不在于谁会背规则，而在于谁更早明白：

- 牌不是一张一张出的。
- 牌权不是一轮一轮算的。
- 升级目标，才是整局里真正该服务的东西。
